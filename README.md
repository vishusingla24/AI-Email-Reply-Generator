# AI Email Reply Generator

An AI-powered email reply generator built using n8n and Google Gemini API.

## Features

* Generates professional email replies automatically
* Uses Google Gemini AI for response generation
* Supports different email types such as job applications, support requests, and general inquiries
* Built using no-code/low-code workflow automation

## Tech Stack

* n8n
* Google Gemini API
* Prompt Engineering
* Workflow Automation

## Workflow

Input Email → Gemini AI → Professional Email Reply

## Sample Use Case

Input:
"Hello, I would like to know the status of my application."

Output:
A professional and context-aware email response generated by Gemini AI.
